**Installation**
################

Once you've cloned or put the vfxAssetBox somewhere on your local machine,
add the path to vfxAssetBox to your PATH or PYTHONPATH env variable.

eg:
PYTHONPATH G:/Scripts/VfxAssetBox

**Running**
###########

.. code-block:: python

    #To launch in a PySide compatible application, run this command:
    import assetbox
    assetbox.launch()

To execute it directly, run as such:

.. code-block:: python

    execfile('c:/path/to/vfxAssetBox/assetbox/ui/launcher.py')
